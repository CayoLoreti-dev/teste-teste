# Catálogo SSISSI

Site de catálogo de vestidos com experiência leve e elegante, reunindo coleções para **noivas**, **formaturas** e **madrinhas**. O foco é apresentar modelos de forma clara, com navegação simples e visual consistente em todas as páginas.

## O que o site entrega

- **Página inicial** com chamada para os catálogos.
- **Catálogos separados** por ocasião: noiva, formatura e madrinha.
- **Visual coeso** com estilos reaproveitados.
- **Componentes compartilhados** (header, login e modais) para manutenção rápida.

## Como acessar

- Abra o arquivo `index.html` no navegador.
- Para evitar problemas de caminho, sempre abra pela **raiz do projeto**.

## Publicação

O site já está pronto para hospedagem estática. Se usar Netlify, basta apontar o repositório e publicar a raiz (`.`). O `netlify.toml` já está configurado.
